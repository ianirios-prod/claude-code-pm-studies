# Claude Code — PM Learning Repository

Este repositório é um **guia de aprendizado** para usar Claude Code como Technical AI PM.

## O que tem aqui

- `learning-guide/` — Conceitos, mental models, tips & tricks do Claude Code para PMs
- `.claude/commands/` — Templates de commands reutilizáveis para tarefas de PM

## Para que serve

Este repositório captura o conhecimento sobre **como usar Claude Code** como ferramenta de trabalho para PMs técnicos. Não tem dados de nenhuma empresa real ou fictícia — é puramente sobre a ferramenta.

## Repositório de prática

Para praticar com arquivos realistas de PM (entrevistas, PRDs, métricas, etc.), use o repositório companion:
→ `claude-code-pm-simulation` (DataFlow AI — empresa fictícia para prática)

## Regras para Claude Code

- Respostas em português, exceto se o output for documento técnico
- Quando sugerir usar um command, mostrar o nome exato do command (ex: `/prd`)
- Manter outputs concisos e acionáveis
